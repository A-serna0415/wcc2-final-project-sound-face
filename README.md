# SOUND FACE
### By Andrés Serna

This project is an interactive computer vision system built with p5.js, MediaPipe, and the p5.sound library. 
It translates facial gestures into a performative audiovisual instrument. Using MediaPipe, the system detects a face
via webcam and extracts the coordinates and relative distances of facial landmarks. These measurements are mapped to 
sound parameters through p5.sound. Visually, the instrument is represented as a particle system that responds dynamically
to the audio generated by specific facial gestures.

How to Run:

- Important to use de browsers Chrome or Firefox.
- To run it locally: go to Terminal en execute the line "node app.js", open server "http://localhost:3000".
- Make sure you give the corresponding permission to the browser to use your webcam. 
- Click/pause screen to start audio output.
- Experiment with the instrument using your face: Smile, frowns, act sad, surprised, tilt your head or try a combination of all of them.

Or experiment the project online: https://sound-face-7g1j.onrender.com
